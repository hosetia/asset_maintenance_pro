# Asset Maintenance Pro - Flutter App

تطبيق موبايل احترافي لإدارة صيانة الأصول والمعدات.

## المميزات

✅ تسجيل دخول آمن
✅ إدارة الأصول
✅ طلبات الصيانة
✅ رفع الصور
✅ الإشعارات
✅ الملف الشخصي

## متطلبات التطوير

- Flutter 3.10+
- Dart 3.0+
- Android SDK 21+
- iOS 11.0+

## التشغيل

```bash
flutter pub get
flutter run
```

## البناء

### APK
```bash
flutter build apk --release
```

### iOS
```bash
flutter build ios --release
```
